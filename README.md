Minimal API Project

Este é um projeto de API desenvolvido como parte do desafio prático de formação em C#/.NET. O objetivo foi estruturar uma API funcional utilizando o padrão de Minimal APIs, acompanhada de testes unitários para garantir a qualidade do código.

🛠 Tecnologias Utilizadas
Linguagem: C#

Framework: .NET 9.0

Banco de Dados: Entity Framework Core (SQLite)

Testes: MSTest

🚀 Como Executar
Clone este repositório:
git clone [https://github.com/ThiagoTasi/minimal-api]

Acesse a pasta do projeto:
cd MINIMAL-API/API

Restaure as dependências:
dotnet restore

Execute a aplicação:
dotnet run

🧪 Testes
Para rodar os testes unitários, na raiz do projeto (MINIMAL-API), execute:

Bash
dotnet test
